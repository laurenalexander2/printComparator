A computer-vision tool for comparative analysis of print impressions.

Compare two digitized impressions from the same intaglio plate (etching, engraving, drypoint, mezzotint, and related processes). The tool reduces digitization variability, registers impressions into a shared coordinate space, and produces outputs designed for interpretation.

How it works

1. Preprocess for comparability
Convert to grayscale, standardize scale, and normalize contrast to reduce exposure differences, paper tone, and capture artifacts—so the comparison is driven more by print variation than digitization quirks.
2. Align one impression to the other
Detect shared landmarks, estimate the transform, and warp the target impression into the base impression’s coordinate space. If feature matching is weak or unstable, the alignment falls back to a gentler global method.
3. Visual analysis (two modes)
A.Pixel-level absolute difference: Map Best when changes are strictly additive (added tone, heavier inking, strengthened marks), because magnitude reads cleanly.
B. Color Overlays: Tint each impression (red vs cyan) so regions that agree collapse toward neutral/darker values, while differences remain strongly colored. Best for rapid scanning and localizing change—especially when differences are mixed.
4. Interactive controls
Browser UI: file upload, sliders/toggles, session state, and real-time visualization of aligned outputs and analysis views.






The Tech Stack

Python-based image processing with an interactive Streamlit UI.

OpenCV (cv2)
Preprocessing (grayscale, resize, blur, histogram equalization), feature alignment (AKAZE + BFMatcher + Lowe ratio), robust transform estimation (RANSAC homography), warping/registration (perspective + affine), and difference representations (absdiff, thresholding, morphology, connected components, overlay composition).
ECC affine fallback
When homography estimation fails validation or inputs are low-feature, ECC optimization finds a stable affine alignment (rotation/scale/translation). This avoids degenerate warps and degrades gracefully.
NumPy
Vectorized pixel arithmetic for masks and overlays, fast threshold logic, and feature vectors derived from change masks (grid ratios + global ratios).
Streamlit
Browser UI: file upload, sliders/toggles, session state, and real-time visualization of aligned outputs and analysis views.
scikit-learn (KMeans, optional)
Experimental clustering of impressions using feature vectors derived from change masks to explore hypothesized grouping/state structure.
